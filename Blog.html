Anomaly Detection-Using Voting Algorithms

There are Different Transaction Fraud Occur in our Daily life.One of the Category of Transacation is Credit Card Fraud Transaction.Fraudster Steal the Details of Credit card of an Individual via phishing,skimming etc kind of Techniques.we aim to Build a Machine Learning model which can detected whether the given transaction is Fraudalent or Not on the Basis of Previous Transaction Behaviour of the Card Holder.We train a Machine Learning Model with Transaction dataset,The model Comprises of different types of Classification Algorithms which can Classify Fraudalent and Fair Transaction.This System can be used as an Integrated Software in Online payement Gateways,Credit Card Swipe Machines etc which can help customer by saving Him with Transaction Fraud's.
